## 图标获取方式

### 方式一：复制 `drawable-anydpi-v24` 文件

Material Design Icons 项目已经为我们准备了适配 Android 的矢量图标文件，位于：

```
E:\Android\Material Symbols\material-design-icons\android\[类别]\drawable-anydpi-v24\ic_[图标名].xml
```

### 方式二：使用 `outlined` 样式

部分图标有单独的 `outlined` 样式文件夹：

```
E:\Android\Material Symbols\material-design-icons\android\[类别]\res\drawable-anydpi-v24\ic_[图标名]_outline.xml
```

## 复制命令参考

以下命令可用于批量复制图标（需根据实际路径调整）：

```bash
# 示例：复制 action 类别图标
cp "E:/Android/Material Symbols/material-design-icons/android/action/drawable-anydpi-v24/ic_search.xml" "E:/Android/SafeVault/app/src/main/res/drawable/"

# 复制 navigation 类别图标
cp "E:/Android/Material Symbols/material-design-icons/android/navigation/drawable-anydpi-v24/ic_arrow_back.xml" "E:/Android/SafeVault/app/src/main/res/drawable/"

# 复制 social 类别图标
cp "E:/Android/Material Symbols/material-design-icons/android/social/drawable-anydpi-v24/ic_person.xml" "E:/Android/SafeVault/app/src/main/res/drawable/"
```

## 需要注意的图标

以下图标在当前项目中可能需要特殊处理：

| 图标 | 问题 | 建议 |
|------|------|------|
| `ic_settings_appearance.xml` | Material Design 无直接对应 | 使用 `palette` 或 `style` |
| `ic_back.xml` | 与 `ic_arrow_back` 重复 | 考虑删除或复用 |
| `ic_password.xml` | 与 `ic_lock` 重复 | 保持一致或创建独特设计 |
| `ic_safe_vault.xml` | 品牌图标 | **保留原设计**或使用 `vault`（如有）|
| `ic_launcher_foreground.xml` | 应用图标 | **保留原设计** |
