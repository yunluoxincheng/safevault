## 🔐 安全性说明

### 端到端加密
- 使用 RSA-2048 位密钥加密
- 只有接收方可以解密
- 服务器无法访问明文密码
- 零知识架构

### 隐私保护
- 不收集位置信息
- 联系人关系不上传
- 分享记录加密存储
- 主密码派生所有密钥

## 🎯 功能亮点

### 1. 联系人管理
- 扫描二维码快速添加
- 手动输入身份码
- 为联系人添加备注
- 安全存储公钥

### 2. 密码分享
- 选择接收方
- 设置权限（查看/保存/撤销）
- 设置过期时间
- 生物识别验证

### 3. 分享接收
- 扫描二维码接收
- 离线文件支持
- 保存到密码库
- 查看分享详情

### 4. 分享历史
- 查看所有分享
- 撤销活跃分享
- 查看分享状态
- 管理收到的分享
