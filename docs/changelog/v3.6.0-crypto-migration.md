# SafeVault v3.6.0 发布说明

## 版本信息

**版本**: 3.6.0
**发布日期**: 2026-03-03
**兼容性**: Android 10+ (API 29-36)

---

## 🎉 主要更新

### 🚀 密码分享性能大幅提升

**引入现代椭圆曲线加密算法 X25519/Ed25519**

| 操作 | v2.0 (RSA) | v3.0 (X25519/Ed25519) | 提升 |
|------|-----------|---------------------|------|
| 密钥生成 | ~50ms | ~1ms | **50x** |
| 加密/解密 | ~50ms | ~1ms | **50x** |
| 签名 | ~10ms | ~0.1ms | **100x** |
| 公钥大小 | 256B | 32B | **8x** |
| 私钥大小 | ~1.2KB | 32B | **37.5x** |

### 🔒 安全增强

- **前向保密**: 每次分享使用新的 ephemeral key，长期私钥泄露不影响历史分享
- **身份绑定**: HKDF 密钥派生混合双方身份，防止密钥混淆攻击
- **防重放攻击**: v3.0 包含时间戳验证和过期检查
- **Invalid Curve 防护**: 验证 ephemeral public key 有效性

### 🤖 智能版本协商

系统会自动检测接收方支持的协议版本：
- 双方支持 v3.0 → 使用 **X25519/Ed25519**（最优）
- 一方仅支持 v2.0 → 自动**回退到 RSA**（兼容）

### 📦 优化内容

- ProGuard 配置优化，减小 APK 体积
- 后端集成测试覆盖更全面
- 性能测试套件验证加密算法速度

## 🔧 迁移说明

### 用户操作

1. 打开 SafeVault 应用
2. 进入「设置」页面
3. 找到「加密算法」选项
4. 点击「迁移到新加密算法」
5. 输入主密码验证
6. 等待迁移完成（< 1 秒）

### 迁移特性

- ✅ **安全**: 不删除 RSA 密钥，可安全回滚
- ✅ **快速**: 通常 < 1 秒完成
- ✅ **幂等**: 失败后可重试
- ✅ **无影响**: 密码数据和分享链接不受影响
